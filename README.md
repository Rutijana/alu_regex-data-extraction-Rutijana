# Data Extraction with Regular Expressions

## Overview

This project demonstrates how to extract various types of data from a string using Python's regular expressions (regex). The data includes email addresses, URLs, phone numbers, credit card numbers, time formats, HTML tags, hashtags, and currency amounts.

## Purpose

The purpose of this project is to provide a simple and effective way to extract structured information from unstructured text data. 

## Features

The script extracts the following data types:

1. **Email Addresses**: Matches standard email formats.
2. **URLs**: Matches web addresses starting with `http://` or `https://`.
3. **Phone Numbers**: Matches various phone number formats.
4. **Credit Card Numbers**: Matches credit card numbers with spaces or hyphens.
5. **Time Formats**: Matches both 12-hour and 24-hour time formats.
6. **HTML Tags**: Matches any HTML tags present in the text.
7. **Hashtags**: Matches hashtags that start with `#`.
8. **Currency Amounts**: Matches currency amounts with optional commas and decimal points.
